BounceBack

BounceBack is a full-stack messaging app that allows many users to chat together in real time. The backend is built using Ruby on Rails with a PostgreSQL database. The frontend is built on Facebook's ReactJS using Flux architecture and the design is done using Google's Material Design Lite.

Maze Solver

Maze Solver is a JavaScript app that generates and solves mazes. First, mazes are generated using a random Prim's algorithm. Then, the shortest path between two, user specified, points is found using the A* algorithm. This priority queue used in A* is implemented using a hand made Fibonacci heap.
Live GitHub

Snails

Snails is light weight Ruby web framework based on the MVC pattern. It was developed with Matt Fong, Bryan Ng and Ben Calabrese. Though originally a project to better understand how web frameworks work under the hood, it is now fully functional web framework that that has been used to create several small web apps.

SKTest

SKTest is a Python integration testing suite I built while working at Paulson Traing Programs, Inc. (no relation). I was working on a web app version of their SkillBuilder training software and wrote a testing suite using the Selenium WebDriver testing library. SKTest includes a custom testing DSL allowing non-technical users to write tests as CSV files.
